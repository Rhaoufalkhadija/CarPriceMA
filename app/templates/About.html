À propos du projet

CarPriceMA est une application web développée pour prédire le prix des voitures d'occasion au Maroc en utilisant des techniques de Machine Learning.

Objectif :

Fournir aux utilisateurs une estimation rapide et fiable du prix de leur véhicule basé sur des critères tels que la marque, le kilométrage, l’année, etc.

Technologies utilisées :

  • Python + Flask pour le backend
  • HTML/CSS/ Javascript + Bootstrap pour le frontend
  • Modèle de Machine Learning (Regression / RandomForest)

Données :

Le modèle a été entraîné sur un dataset de voitures d’occasion marocaines collecté en ligne (par exemple : Avito.ma).

Développeuse :

Ce projet a été réalisé dans le cadre d’un projet académique de Machine Learning.
